# QualityMate

Self-hosted ISO 9001 quality management for construction firms. Lives on your hardware. Your data never leaves.

## What it does

- Daily site check-in (QR poster per job).
- Incident logging with photos, AI-assisted triage, corrective actions.
- Quarterly ISO 9001 management-review meetings — pack, sign-offs, minutes, distribution.
- Audit-log everything. Backups every night. Restore in one command.

## What it isn't

- Not a SaaS. No multi-tenant cloud. One install per company.
- Not a substitute for an ISO 9001 lead auditor — it's the system *they* will audit.
