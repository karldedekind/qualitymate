Common questions

What happens if I stop paying support?

Nothing — your install keeps running on the version you have. You just don't get patches or upgrades until you renew.

Is there a trial?

Email us and we'll set up a 14-day demo install on our infrastructure. After that, the licence is required to keep running.

Can I run it on my own hardware?

Yes — that's the whole point. Any Linux host with Docker. 2 GB RAM, 2 vCPU, 20 GB disk minimum.

Multi-company or multi-tenant?

No. One install per company. If you operate multiple trading entities and need separate audit trails, run multiple installs.

Do you take credit cards?

EFT only for the licence. Support and white-glove can be invoiced. Talk to us.
